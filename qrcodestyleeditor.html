QRCode Style Editor

This application, developed using Python and the PyQt5 library, allows for extensive customization of QR Codes. Users can modify the shape, colors, and apply color gradients to create unique QR Code designs.

This application generates QR Codes from user input data. Additionally, it allows customization options such as adjusting error correction levels and embedding an image in the center of the QR Code.

It also includes a section for decoding QR Codes, allowing users to extract and view the embedded text data.
